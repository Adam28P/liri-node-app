<h1>Liri Bot App</h1>
<h3>LIRI is a command line node app that takes in parameters and returns data based on one of four commands:
<ul>
<li>concert-this</li>
<li>spotify-this-song</li>
<li>movie-this</li>
<li>do-what-it-says</li>
</ul>
</h3>
